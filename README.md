# kiedywakacje.com

## Opis
Serwis pokazujący daty wakacji szkolnych, świąt i dni wolnych w Polsce.  
Celem projektu jest szybkie i proste sprawdzenie, ile dni zostało do najbliższego odpoczynku.

## Użytkowanie
Wejdź na [kiedywakacje.com](https://kiedywakacje.com) – daty i licznik pokażą się automatycznie.

## Technologie
- Next.js
- TailwindCSS


## Uruchamianie lokalnie
```bash
git clone https://github.com/twoje-repo/kiedywakacje.com.git
cd kiedywakacje.com
npm install
npm run dev
```

#whenisbreak.com - kiedywakacje.com
Description

A website that shows school holiday dates, public holidays, and days off in Poland.
The goal of the project is to let users quickly check how many days are left until the next break.

Usage
```bash
git clone https://github.com/twoje-repo/kiedywakacje.com.git
cd kiedywakacje.com
npm install
npm run dev
```
Visit kiedywakacje.com.com
 – dates and countdown are displayed automatically.

Technologies
Next.js
TailwindCSS


Last time using animated SVG backgrounds. Performance is basically suffering for no good reason.
The design doesn’t really match what I originally had in mind, and the responsiveness still needs work in multiple places.

Right now it functions, sure, but “works” is a low bar that even broken microwaves clear.

The project needs a stronger design direction and proper UI consistency instead of just stacking features and hoping it magically becomes good.

TL;DR

Works fine, looks mid, performs worse than it should, needs redesign and better responsive layout decisions.

And just to ruin the illusion of progress: the biggest assumption here is that “it works so it’s fine.” That’s exactly how you end up shipping technically correct but unpleasant products that nobody remembers five minutes later.
