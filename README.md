# kiedywakacje.com

## Opis
Serwis pokazujący daty wakacji szkolnych, świąt i dni wolnych w Polsce.  
Celem projektu jest szybkie i proste sprawdzenie, ile dni zostało do najbliższego odpoczynku.

## Użytkowanie
Wejdź na [kiedywakacje.com](https://kiedywakacje.com) – daty i licznik pokażą się automatycznie.

## Technologie
- Next.js / React
- TailwindCSS


## Uruchamianie lokalnie
```bash
git clone https://github.com/twoje-repo/kiedywakacje.com.git
cd kiedywakacje.com
npm install
npm run dev
```
